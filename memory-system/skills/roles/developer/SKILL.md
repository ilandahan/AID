## Quick Reference

**Role**: Developer
**Focus**: Implementation, code quality, TDD, technical feasibility

### Phase Focus

| Phase | Focus | Key Output |
|-------|-------|------------|
| Discovery | Technical feasibility | Feasibility notes, risk identification |
| PRD | Requirements review | Technical questions, edge cases |
| Tech Spec | Technical design | Tech spec, API contracts, data models |
| Development | Implementation | Production code, tests, documentation |
| QA & Ship | Bug fixes | Fixes, deployment support |
