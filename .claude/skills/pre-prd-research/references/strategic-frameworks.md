# Strategic Frameworks Reference

Complete guide to strategic analysis frameworks for product positioning and opportunity identification.

## Blue Ocean Strategy

Use this to create uncontested market space:

## SWOT Analysis

### Template

```markdown
## SWOT Analysis: [Subject]
**ID**: [PROJECT]-C-SWOT-XXX
**Date**: [Date]

### Internal Factors

| Strengths (Leverage) | Weaknesses (Address) |
|---------------------|---------------------|
| S1. [Strength] | W1. [Weakness] |
| S2. [Strength] | W2. [Weakness] |
| S3. [Strength] | W3. [Weakness] |
| **Research**: [IDs] | **Research**: [IDs] |

### External Factors

| Opportunities (Pursue) | Threats (Mitigate) |
|-----------------------|-------------------|
| O1. [Opportunity] | T1. [Threat] |
| O2. [Opportunity] | T2. [Threat] |
| O3. [Opportunity] | T3. [Threat] |
| **Research**: [IDs] | **Research**: [IDs] |

### Strategic Options (TOWS Matrix)

| | Strengths | Weaknesses |
|-|-----------|------------|
| **Opportunities** | **S-O Strategies** (Leverage) | **W-O Strategies** (Overcome) |
| | Use S1 to capitalize on O1 | Address W1 by pursuing O2 |
| **Threats** | **S-T Strategies** (Defend) | **W-T Strategies** (Avoid) |
| | Use S2 to counter T1 | Minimize W2 exposure to T2 |
```

## Opportunity Scoring Matrix

### Scoring Criteria

```
┌─────────────────────────────────────────────────────────────────┐
│                 OPPORTUNITY SCORING MATRIX                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  SCORE = (Impact × Confidence) - (Effort × Risk)                │
│                                                                  │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │ IMPACT (1-10)                                               │ │
│  │ • User value: Does it solve a real pain?                   │ │
│  │ • Business value: Revenue/retention impact?                │ │
│  │ • Strategic value: Differentiation/positioning?            │ │
│  └────────────────────────────────────────────────────────────┘ │
│                                                                  │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │ CONFIDENCE (1-10)                                           │ │
│  │ • Evidence quality: How validated is this?                 │ │
│  │ • User demand: Clear pull vs. push?                        │ │
│  │ • Technical feasibility: Can we build it?                  │ │
│  └────────────────────────────────────────────────────────────┘ │
│                                                                  │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │ EFFORT (1-10)                                               │ │
│  │ • Development: How long to build?                          │ │
│  │ • Dependencies: External blockers?                         │ │
│  │ • Complexity: Technical difficulty?                        │ │
│  └────────────────────────────────────────────────────────────┘ │
│                                                                  │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │ RISK (1-10)                                                 │ │
│  │ • Technical risk: Unknown territory?                       │ │
│  │ • Market risk: Will users adopt?                           │ │
│  │ • Competitive risk: Easily copied?                         │ │
│  └────────────────────────────────────────────────────────────┘ │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## When to Use Each Framework

| Framework | Best Used For | Key Output |
|-----------|--------------|------------|
| Blue Ocean | Finding uncontested market space | Four Actions + Value Curve |
| Value Proposition Canvas | Ensuring product-market fit | Fit Score + Gaps |
| SWOT | Quick situational assessment | Strategic options |
| Porter's Five Forces | Industry attractiveness analysis | Force map + positioning |
| Opportunity Scoring | Prioritizing features/initiatives | Ranked list |
| Strategy Canvas | Competitive differentiation | Value curve comparison |

## Integration with Traceability

All strategic framework outputs should be assigned IDs:

- `[PROJECT]-C-BLUE-XXX` - Blue Ocean analysis
- `[PROJECT]-C-VPC-XXX` - Value Proposition Canvas
- `[PROJECT]-C-SWOT-XXX` - SWOT analysis
- `[PROJECT]-C-5F-XXX` - Five Forces analysis
- `[PROJECT]-C-SCORE-XXX` - Opportunity scoring
- `[PROJECT]-C-CANVAS-XXX` - Strategy canvas

These IDs link to opportunities (`C-OPP-XXX`) which feed into PRD requirements.
